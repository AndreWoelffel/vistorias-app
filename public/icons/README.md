# Ícones PWA

Arquivos esperados:

- `icon-192.png`, `icon-512.png` — manifest (`purpose: any`)
- `icon-512-maskable.png` — **maskable** (área segura; substitua por arte final 512×512)
- `apple-touch-icon.png` — iOS (ideal: 180×180 ou 192×192)
- `favicon-32.png` — aba do navegador
- `icon.svg` — opcional (referência visual)

Execute `node scripts/create-pwa-placeholder-icons.mjs` para gerar PNGs mínimos válidos se ainda não existirem, ou substitua por PNGs profissionais mantendo os nomes.
